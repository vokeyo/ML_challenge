Out-of-Date Questionnaires and Related Materials

The NCI strongly discourages the use of the 1987 Block-NCI HHHQ questionnaires to collect current or future dietary data. Although these questionnaires have been validated for dietary data collected in the 1980's and early 1990's, they have not been updated and re-evaluated for current use. These questionnaires and associated DietSys files are being made available here as a resource for investigators with active studies using the 1987 questionnaires. If you are initiating a study, please use a modified version of the 1992 Block-NCI questionnaire or a recent FFQ available from other institutions or researchers.

This zip file also includes PDF files containing the codebooks for the 1987 questionnaires. Save the file to your hard drive in the same directory as the DietSys program. Run the DIET87.EXE program to extract the files. See DIET87.TXT for a description of the DIET87 files.

Interactive Interview Software

Previous versions of the DietSys software included an interactive interview feature. This feature had been included for investigators who had used the interview for studies in the 1980's. It is strongly recommended that you do not use this interview. It is an outdated questionnaire and the software does not allow you to make significant modifications. However, we are making this available as a stand-alone program (DietInt) for investigators with existing studies.

The files required for the DietInt are contained in the self-extracting zip file below. Save this file to your hard disk. It may be saved in the same directory as the DietSys program. Run INTERVW.EXE to extract the DietInt program files. Run the interviews using DIETINT.EXE. See the DIETINT.TXT and INTERVW.INS text files included with the DietInt program files for more information.
