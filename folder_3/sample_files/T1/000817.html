A situation in which one (1) or more support orders have been issued for the same parties. Rules were established under UIFSA to determine which single order is the controlling order and is the one (1) order which must be honored and recognized.

Acronym for the Uniform Interstate Family Support Act.

U27.5 Interstate Cases

This policy will be applied to all cases with debts based on Idaho issued orders.  This includes Interstate cases (both incoming and outgoing) that contain current support debts based on Idaho issued orders.

This policy may apply to non Idaho issued orders.  If the debt is based on a non Idaho issued order, the Case Manager will need to contact the issuing state to verify that state’s laws, policy and/or procedures.  If the issuing state allows the accrual to be stopped in these situations, the CSS Case Manager will follow the laws, policy and/or procedures of the issuing state.   

U27.5.1 Multiple Orders Issued by Idaho and/or Another State.

Whenever multiple orders exist, Continuing Exclusive Jurisdiction (CEJ) and Determination of Controlling Order (DCO) will need to be made to determine if this policy can be applied to the current support accrual. 

  • If the multiple orders are all issued by Idaho courts, DCO will be determined by applying Idaho’s intrastate policy.  Because the multiple orders are all issued by Idaho courts, this policy will be applied to determine how marriage, re-marriage, and/or co-habitation affect current support accrual.

  • If one of the orders is issued by Idaho and one is issued by another state, CEJ and DCO will be determined using UIFSA Interstate policy found in Chapter 8.  If the Idaho order is determined to be the Controlling Order, this policy will be applied to determine how marriage, re-marriage, and/or co-habitation affect current support accrual.

  • If one of the orders is issued by Idaho and one is issued by another state, CEJ and DCO will be determined using UIFSA Interstate policy found in Chapter 8.  If the other states’ order is determined to be the controlling order, this policy will not be applied. The CSS Case Manager will contact the other state to determine how, or if, marriage, re-marriage, and/or co-habitation affects current support accrual. Idaho will defer to the laws and policies of the state that issued the controlling order.

  • If none of the multiple orders are issued by Idaho, CEJ and DCO will be determined using UIFSA Interstate policy found in Chapter 8. Since none of the orders were issued by Idaho, this policy will not be applied. The CSS Case Manager will contact the state that issued the controlling order to determine how, or if, marriage, re-marriage, and/or co-habitation affects current support accrual.
