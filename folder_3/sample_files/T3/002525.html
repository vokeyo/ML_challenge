Goals & Features
  The purpose of this site is to provide a searchable database of STEREO journal publications. The database is a beta so please be patient and send any comments or suggestions to the feedback address. At the moment entries are being added by the webmaster so any additions should be sent to that address. Some features are not yet completely implimented and therefore may not work properly.

About
  This literature database is maintained by the STEREO Science Center (SSC). You're welcome to send any questions or suggestions to our feedback address. The database is powered by refbase, an open source database front-end for managing scientific literature & citations.
