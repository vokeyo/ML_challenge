Using the Spelling Corrector

 
 

TotalView can check the spelling of text entries for certain commands. If TotalView does not find the name you entered, it displays a dialog box with the closest match, as shown in the following figure.

Spelling Corrector Dialog Box

You can edit the closest match, and then select OK to use it, Original to get back the original text, or Abort to cancel.

To customize the behavior of the spelling corrector with X Window System resources, refer to totalview*spellCorrection.
