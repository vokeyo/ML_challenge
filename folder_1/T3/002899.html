Cómo Bajar Datos de Mapas Temáticos

Los dos formatos de archivos son compatibles con bases de datos, es decir, sólo se bajan las filas de datos.

  • Microsoft Excel - este es el formato de archivo compatible con hojas de cálculos (.xls file).
  • Base de datos delimitados por comas - este formato de archivo es para bajar los récords de datos para poder cargarlos al programa de computadora de base de datos para manipular los datos (.txt file)
Los archivos bajados compatibles con bases de datos se comprimen en un solo archivo titulado output.zip. Este archivo comprimido contiene:

  • Un archivo de datos- este archivo contiene las filas de datos para las Tablas de Comparación Geográfica que usted ha seleccionado. Cada fila de datos contiene las partidas de datos y la siguiente información geográfica:
    • Identificador Geográfico (GEO_ID)
    • Nombre de Ruta (PATH_NAME)
    • Código del Componente Geográfico (GEOCOMP)
    • Códigos FIPS y nombres geográficos específicos para las selecciones geográficas para los mapas (por ejemplo, Puerto Rico por municipio, municipio por sector censal, etc.)
  • Datos de Mapa (MAP_DATA) - el valor de los datos usado para crear el mapa de las características
  • Clase de Mapa (CLASS) - el grupo numérico al cual se clasifica el valor de datos en el mapa
  • Un archivo meta.txt - este archivo explica la formula para crear el mapa, el tipo de la clase de mapa y las divisiones en clases.
  • Un archivo tm_readme.txt- este archivo explica el contenido de los archivos bajados, las anomalías específicas al conjunto de datos y la localización de la documentación específica relacionada con el conjunto de datos.
  • Un archivo readme.txt específico para el conjunto de datos - este archivo explica porque puede que no haya datos para algunas selecciones geográficas. Nota: Los Mapas Temáticos sólo pueden mostrarse y bajarse uno a la vez.

    Casilla de cotejo Opciones "Activada" (predeterminada) significa:

    Incluya el nombre descriptivo largo en el archivo bajado. Cada elemento de datos para cada tabla tiene un nombre corto en clave y un nombre descriptivo largo. Por ejemplo:

    • Nombre corto en clave: P007006 donde P007 es el número de tabla y 006 es el número de elementos de datos en esa tabla del cualse deriva el valor de MAP_DATA.
    • Nombre descriptivo largo: Total
    Casilla de cotejo Opciones "Desactivada" significa:

    • Sólo aparecerá el nombre en clave en el archivo bajado
    CONSEJOS para usar el formato compatible con bases de datos:

    • Puede guardar output.zip con ese nombre o con otro de su creación.
    • Asegúrese de consultar el archivo readme.txt para explicación de todos los tipos de archivos que ha bajado.
