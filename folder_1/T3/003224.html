Content Description

Abstract: This data depicts the coastline of New Jersey. It was interpreted using 1986 JSS CIR photography, using Baush Lomb 95 stereoscope to mylar overlays to the 1986 photo basemap. This line data is unattributed.
Purpose: This data was produced to provide a delineation of the coastline of New Jersey in a graphical format.
