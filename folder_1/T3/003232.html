Viking Orbiter Stereo Images of the Mars Pathfinder Landing Site


Viking 004a Stereo Locations

Processing by Tim Parker (timothy.parker@jpl.nasa.gov)
Jet Propulsion Laboratory


This page contains a series of Viking Orbiter images from orbit 004A that offer excellent stereo views of much of the Mars Pathfinder landing site ellipse in the Ares Vallis region of Mars (19.5 deg N, 32.8 deg W). These images were map-projected and filtered using VICAR image processing software at the Multimission Image Processing Laboratory at JPL. Supplemental processing to improve image clarity for viewing subtle features on the plains within the landing site was performed on a Macintosh desktop computer using Adobe Photoshop ®. All files are JPEG format. See the JPEG file "Viking 004A Stereo locations" above to determine the placement of each frame within the landing site ellipse.

Each image includes a title block, explaining the map projection (orthoghraphic), the pixel scale of the projection (30m/pixel; original data varied slightly in pixel size around 40m/pixel during the orbit track), the separation angle (so local, relative topographic measurements can be made), and the Picture number (Picno). At the bottom of each frame are a north arrow and scale bar (the frames are oriented for optimal stereo viewing when placed with text up, so north is not straight up), and instructions for placement of each frame for stereo viewing. Each picture is either a "Left" or "Right" eye image (approximately 23 in x 30 in), and conjugate images are indicated by "Pair with." Click on any of the GIF thumbnails below to download the large (500K-600K) corresponding JPEG file. For optimal stereo viewing, the frames should be printed on a photo-quality printer (though high-quality laser and inkjet prints will also work well) and viewed using a desktop stereoscope (some users, however, are able to view stereo images without a stereoscope).
